# 🚀 Karo Pitch - Startup Landing Page

Karo Pitch is a modern, investor-focused startup landing page designed to help founders connect with investors across India through curated demo days.

This project is built using HTML, CSS, and vanilla JavaScript with a strong focus on clean UI, responsiveness, and conversion-driven design.

## ✨ Features

- 🎯 Investor-focused landing page structure
- 📱 Fully responsive (mobile + tablet + desktop)
- 🎨 Premium Bharat-inspired UI design
- ⚡ Smooth scrolling navigation
- 👀 Scroll reveal animations using IntersectionObserver
- 🌈 Subtle gradient background orbs
- 🧩 Clean and reusable code structure

## 🛠️ Tech Stack

- HTML5
- CSS3 (Custom Properties / Variables)
- Vanilla JavaScript

## 🤖 AI Usage

This project was created with the help of ChatGPT, which assisted in:

- Structuring the landing page
- Generating clean and semantic code
- Designing UI sections and layout
- Implementing animations and responsiveness

## 🎨 Design Philosophy

The design follows a warm, premium Bharat-focused startup vibe.

Typography:
- Headings -> Space Grotesk
- Body -> Urbanist

Color Strategy:
- Soft warm backgrounds for comfort
- Strong orange brand color for energy and action
- Teal accent for trust and balance

User Flow:
- Hero -> Clear value proposition
- About -> Context
- How it works -> Process clarity
- Investors and Startups -> Social proof
- CTA -> Conversion
